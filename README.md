AccidentDetection is an AI-powered road accident detection system that uses computer vision to monitor video streams and detect potential accidents in real time. When an accident is detected, the system automatically sends an alert message using Twilio and stores the incident data in MongoDB for future analysis.

Features

Real-time accident detection using computer vision

Automatic SMS alerts using Twilio API

Incident data storage in MongoDB

Monitoring and tracking previous accident events

Tech Stack

Python

OpenCV

Machine Learning

Twilio API

MongoDB
